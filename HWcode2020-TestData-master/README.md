华为codecraft2020模拟测试数据

## 指南

### 使用方法

* /testData目录下为测试数据及其答案，部分测试数据由 @izhangrui 提供

* /java目录下为测试数据生成源码，可通过Config接口配置账户总个数和转账总个数

* /C++目录下cpp版数据生成程序由 @izhangrui 提供

* 官方规则中循环数量不超过300W，推荐边个数设置为28W，顶点个数设置为2.6W
